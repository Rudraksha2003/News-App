# Live News Aggregator

A lightweight, responsive news aggregation web app built using **vanilla HTML, CSS, and JavaScript**.  
It fetches real-time news across multiple categories and focuses on clean UI, usability, and client-side logic.

## Features
- Live news from public APIs
- Categories: Latest, Tech, Cyber, Sports, National, International
- Search with keyword highlighting
- Save/remove articles using localStorage
- Dark/light mode toggle
- Relative publish time (e.g., 2h ago, Yesterday)
- Fallback images for missing thumbnails
- Fully responsive card-based UI

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Public News API
- LocalStorage

## Project Structure
index.html
style.css
app.js


## Setup
1. Clone the repo  
2. Add your API key in `app.js`  
3. Open `index.html` in a browser

## Notes
- Uses free-tier APIs (rate limits apply)
- Frontend-only project (no backend)
- Built for learning and portfolio use
